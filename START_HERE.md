## 💡 **Key Features Explained**

### **Automatic Profit Calculation**
```
Every sale automatically calculates:
Profit = (Selling Price - Cost Price) × Quantity

Example: Rice sold 10 units
Profit = (₦7,000 - ₦5,000) × 10 = ₦20,000 ✅ AUTOMATIC
```

### **Automatic Stock Updates**
```
Sale recorded:     Stock decreases ✅
Purchase recorded: Stock increases ✅
Stock low:         Alert generated ✅
```

### **Smart Forecasting**
```
Uses last 30-60 days of data to predict:
- What will sell (Demand Forecast)
- Revenue expected (Sales Forecast)
- When you'll run out (Stock Forecast)
```
